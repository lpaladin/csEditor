运行环境
Windows7及以上，装有.net Framework 4.0和Visual C++ 2012 运行时

编译运行
使用VS2012打开csEditor.sln，将cEditorCore设为启动项目，然后就可以生成并运行

程序主要功能
*可以在任意位置进行文字的增加删除功能
*可以进行字符串的替换/全部替换，支持两种算法的选择，KMP算法支持通配符的匹配
*支持鼠标拖放操作

操作说明
Ctrl+D 删除当前行
Ctrl+F 查找
Ctrl+S 保存
其他同系统默认操作，如Ctrl+A全选，Ctrl+C复制等等

编辑->查找/替换 或 Ctrl+F可以调出搜索框，“查找下一个”可以找到光标之后第一个匹配的字符，高级中可以选择不同的匹配算法，其中KMP算法支持通配符匹配（*?），“替换”操作将光标之后第一个匹配字符串替换为“替换内容”，全部替换将采用BM算法，将文本中的全部匹配字符替换
格式->行号 隐藏/显示行号
格式->字体->行高 更改行间距
格式->字体->默认颜色 改变字体颜色
格式->字体->选中颜色 改变文本被选中时的颜色
格式->字体->字体 改变字体及字号大小
